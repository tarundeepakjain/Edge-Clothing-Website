# Edge Clothing Website

## Overview
Edge Clothing is a collaborative web development project created as part of an academic semester project.  
The website represents an online clothing platform built using **basic web technologies**, focusing on clarity, responsiveness, and fundamental backend integration.

The project was intentionally developed **without using modern frontend frameworks**, relying only on core concepts to strengthen foundational understanding.

---

## Live Deployment
- **Frontend:** https://sem2-project-muz1.onrender.com  
- **Backend:** https://edge-clothing.onrender.com  

---

## Technologies Used
### Frontend
- HTML  
- CSS  
- JavaScript (DOM Manipulation)

### Backend
- Node.js  
- Basic server-side routing and logic  

---

## Features
- Responsive and user-friendly UI  
- Static product and content pages  
- Client-side interactivity using DOM manipulation  
- Backend support for handling requests  
- Clean separation between frontend and backend  

---

## Project Approach
- No frontend frameworks or libraries were used  
- JavaScript usage was limited to **DOM manipulation and basic logic**  
- Emphasis on understanding **core web fundamentals**  
- Designed to be lightweight and easy to understand
